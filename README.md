# 🎯 EventPro – Event Budget & Resource Management Dashboard

## 📌 Overview

EventPro is a web-based dashboard designed to manage and analyze event budgets, resources, and vendor networks in real time. It replaces traditional spreadsheets with an interactive and data-driven system for efficient event planning.

---

## 🚀 Key Features

### 💰 Financial Management

* Track total budget, expenses, and remaining balance
* Calculate burn rate and projected final cost
* Analyze funding coverage and cash runway

### 📊 Data Visualization

* Interactive charts using Chart.js
* Budget allocation vs utilization insights
* Category-wise expense breakdown

### 🎛️ Event Templates

* Pre-configured setups for:

  * Weddings
  * Concerts
  * Tech Conferences
  * Corporate Events

### 🧑‍💻 Resource Tracking

* Monitor hardware utilization
* Track staffing levels

### 🌐 External Integration

* Fetch vendor/partner data via APIs

### 🌙 UI/UX

* Responsive design with Tailwind CSS
* Multiple themes + dark mode

---

## 🛠️ Tech Stack

* HTML, CSS (Tailwind), JavaScript
* Firebase (Auth + Firestore)
* Chart.js
* ExcelJS
